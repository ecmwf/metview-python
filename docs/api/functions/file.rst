file
************

..  py:function:: file(path)

    Assigns a Metview file handler to a file specified by ``path``. The file handler can be used in file operations implemented by :func:`write` and :func:`append`.
    
    :param path: output file path
    :type path: str
    :rtype: Metview file handler
